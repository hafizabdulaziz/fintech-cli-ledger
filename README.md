
# 💰 Personal Expense Tracker (Python CLI)

A professional CLI-based Personal Expense Tracker built in Python with CSV data persistence and full CRUD operations. Designed to help users manage their finances with ease and precision.

## 🚀 Capabilities
- **Transaction Management:** Add, view, and delete expenses.
- **CSV Storage:** Data is stored in a portable CSV format for easy export and analysis.
- **Category Filtering:** Organize expenses by categories for better budgeting.
- **Summary Reports:** Get instant feedback on your spending habits.
