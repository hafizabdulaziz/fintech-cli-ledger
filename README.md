# 💰 Personal Expense Tracker (Pro Version)

This is a Python-based application to manage daily expenses. 

### 🌟 Features:
- *Add Expense:* Saves items with date and time.
- *View All Records:* Displayed in a clean format.
- *Search:* Find specific expenses by name.
- *Delete Record:* Remove unwanted entries by serial number.
- *CSV Support:* Data is saved in .csv format, compatible with Microsoft Excel.

### 🛠️ Built With:
- Python (os, datetime modules).
